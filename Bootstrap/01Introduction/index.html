🎒Introducción

Estoy importando Bootstrap usando las etiquetas <link> y <script>

Para las Hojas de Estilo en Cascada:
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.0-beta1/dist/css/bootstrap.min.css">
Para el código de Javascript:
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.2.0-beta1/dist/js/bootstrap.bundle.min.js"><script>

También puedo usar el código fuente de ambos archivos

El enlace de descarga para los archivos fuente de Bootstrap viene en un zip y están separados por funcionalidades. Puedo elegir qué archivos utilizar en el proyecto para obtener funcionalidades específicas, o puedo simplemente agregar los archivos bootstrap.css y bootstrap.bundle.js para agregar el kit completo de Bootstrap al proyecto.

Manejador de paquetes

Si se cuenta con un manejador de paquetes como npm, se puede instalar Bootstrap fácilmente como una dependencia del proyecto. Por ejemplo con npm:
npm install bootstrap@5.2.0-beta1

Etiqueta <meta>

La etiqueta <meta> básica para mayor compatibilidad al renderizar en distintos dispositivos es la siguiente:
<meta name="viewport" content="width=device-width, initial-scale=1">
